# misakaPkgs

Repo under major rework.

For now this repo provides some CTF related python modules,
and an overlay to patch them into nixpkgs.

## python packages

- **[FLARE CAPA](https://mandiant.github.io/capa/)**: The FLARE team's open-source tool to identify capabilities in executable files.
- **[ida-hcli](https://hcli.docs.hex-rays.com/)**: A modern command-line interface for managing IDA Pro licenses, downloads, and more.
- **[ida-netnode](https://github.com/williballenthin/ida-netnode)**: Humane API for storing and accessing persistent data in IDA Pro databases.
- **[ida-settings](https://github.com/williballenthin/ida-settings)**: Fetch and set configuration values for IDA Plugins.
- **[idapro](https://pypi.org/project/idapro/)**: IDA Library Python Module.
- **[libdebug](https://libdebug.org/)**: A Python library to debug binary executables, your own way.
- **[pylingual](https://pylingual.io/)**: Python decompiler for modern Python versions.
- **[qiling](https://qiling.io/)**: Qiling Advanced Binary Emulation Framework.
- **[rust-demangler](https://github.com/teambi0s/rust_demangler)**: A package for demangling Rust symbols, written in Python.
